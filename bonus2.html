Introducing Pico

A stupidly simple and blazing fast,flat file CMS.
Making the web easy

download now

Stupidly Simple

Pico makes creating and maintaining a website
as simple as editing text files

Blazing Fast

Pico is seriously lightweight and doesn't use a
database,making it super fast.

No Database

Pico is a "Flat file" CMS,meaning no database woe's.
no MYSQL queries,nothing

Markdown Formatting

Edit your website in your favourite text editor using
simple Markdown formatting

Twig Templates

Pico uses the Twig templating engine, for powerful
and flexible themes

Open Source

Pico is completely free and open source,released
under the MIT license
